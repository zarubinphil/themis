# Команда: /close-case

Закрывает дело: обновляет статус, архивирует черновики, фиксирует итог.

## Использование

```
/close-case {клиент} {дело}
```

---

## Что делает

1. Обновляет `_case.md`:
   - Статус → "Закрыто"
   - Дата закрытия → сегодня
   - Добавляет раздел "Итог дела": результат, решение суда, исполнено/нет

2. Перемещает всё из `03_drafts/` в `04_archive/`

3. Обновляет `cases/_index.md`: статус → "Закрыто"

4. Выводит итог:
```
✓ Дело закрыто: {клиент} / {дело}
  Черновики архивированы: {N} файлов
  Индекс обновлён
  Материалы дела сохранены в: cases/{клиент}/{дело}/
```

## Важно

Закрытое дело не удаляется. Все материалы остаются в `cases/`.
Для поиска по закрытым делам — смотри `_index.md`, колонка "Статус".
