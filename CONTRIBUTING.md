# Вклад в Themis

Рады улучшениям. Правила:

- **Никаких данных клиентов.** Не коммить ничего из `cases/` кроме синтетического
  примера `cases/ivanov-ivan`. PR с реальными ФИО/реквизитами будут отклонены.
- **Логика — в системные пути** (`.claude/`, `scripts/`, `cockpit/`). Данные
  пользователя (`cases/`, `knowledge/`) править не нужно.
- **Local-first.** Извлечение остаётся локальным (Apple Vision / markitdown /
  whisper). Не добавлять облачные OCR/Ollama на основной проход.
- **Стиль:** Python — PEP8; даты `ДД.ММ.ГГГГ`; без буквы «ё» в документах.
- Перед PR прогони `bash install.sh` на чистом клоне — должно ставиться «под ключ».

Идеи и баги — в Issues.
